10,000 codes

With products without Bar codes, it is convenient to use a pricing gun to label them with a four-digit code. As the code alone would be interpreted as a cash tender, 10000 is put before to the code for data entry. Reference to 10000 codes is a convenient mnemonic to remind staff of the need to enter 10000 first.

If a new product is to be created with a 10000 code, enter the full code as a product ID, which includes the initial 10000.

The New product screen has a button that will assign an unused 10000 code.

In the sales screen, press the # key to fill in the 10000 preamble for the 10000 code.
