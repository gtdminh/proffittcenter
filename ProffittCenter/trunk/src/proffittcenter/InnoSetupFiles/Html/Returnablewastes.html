Returnable wastes

Returnable wastes are products that are not fit for sale but for which a full credit may be claimed from the manufacturer. When products are returned, the stock levels are adjusted.

To Return stock, when products are not fit for sale but will subsequently be credited by a supplier, sell as normal - multiple products or different products can be wasted at one time - and then complete the sale by scanning the returnable wastes bar code or type 1000032 and the Enter key. A keyboard shortcut of ? enters the number 100003; typing a 2 followed by Tab completes the code for return.

The bar code for returnable wastes can be printed out using the  option Print barcodes on the File menu..

Refunds can be in cash or by debit/credit card.
