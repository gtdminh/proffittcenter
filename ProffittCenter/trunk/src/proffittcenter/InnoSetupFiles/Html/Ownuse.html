Own use

For products taken for the retailer's own use, sell as normal, and then scan the own use barcode or type 1000036 followed by the Enter key. Own use purchases are charged for at cost in the Takings screen, and no payment is expected to be made at the time.

Own use purchases are assigned to a particular customer, who, to distinguish him or her from other customers, has an Account name (not customer name) of  'Own use'.

The bar code for own use can be printed out using the  option Print barcodes on the File menu. A keyboard shortcut of ? enters the number 100003; typing a 6 followed by Tab completes the code for own use.

Note that the net figure in the Takings report includes own use purchases at cost.
