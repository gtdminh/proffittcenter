Product

Product details are selected by left clicking the menu item Tables>Product with the mouse.

Locate the required product by typing or scanning the product's bar code or by typing in three or more letters of the product description and selecting with a mouse click, using the Alpha lookup screen. Details of the product can then be changed such as the product description and then clicking Update. Clicking the Price change button can modify the selling price of a product. The details of the SKU that is linked to the product can also be modified by clicking on the SKU button. These details are the entries below the line in the Product screen.

Pressing the Print key or pressing the Print shelf edge label will print out a shelf edge label. There is a small check box next to the button; clicking this will put a barcode on the label. Pressing the Print bar code button will print out an adhesive bar code label on an attached bar code printer selected in the  screen.

A useful technique when a stock code has been wrongly assigned is to change the stock code to the code for a temporary product. This can be done easily by clicking the Unlink button. A new stock code can then be assigned by using the Default stock screen.

It is not possible to delete a product, or modify a product ID once it has appeared is the sales table because of the relational integrity required by the database.
