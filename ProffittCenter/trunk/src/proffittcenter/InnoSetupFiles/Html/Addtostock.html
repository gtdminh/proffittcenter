Add to stock

The Add to Stock screen is used to add products to the system where there is no purchase, such as free stock from visiting reps or replacement for faulty stock.

This screen should never be used for general stock entry; instead use the Delivery screen, so that stock levels are checked, and wholesale price, pack size and supplier are recorded.

The Add to stock screen is activated from the menu item Stock>Add to existing stock. Just scan or type in the bar code, or begin typing the name of the product and use the Alpha lookup procedure.

If the product is recognised, the product description will appear and the focus moves to the Quantity box. Type in the quantity, press Enter to accept. The stock levels will be amended. If the product is not recognised the focus returns to the Product ID box and the system issues a long beep.

The screen is closed by pressing the Cancel button or by using the Escape key.
