Stock value

This screen is activated by the menu command Stock>Stock value>All. There are two buttons â€“ Our stock and Other's stock. Clicking Our stock with the mouse will start the calculation of the total stock value at wholesale price without VAT. The displayed table can be used to identify gross errors - find which department is affected and then use the menu option Stock>Stock value>Department to see which individual stock item is causing problems. The Other's stock button performs the same calculation on stock which is still owned by a supplier.

Cells in the displayed table may be selected with the mouse and then copied to the clipboard by the labelled button. They may then be pasted into a spreadsheet program such as Excel got further analysis or display.
