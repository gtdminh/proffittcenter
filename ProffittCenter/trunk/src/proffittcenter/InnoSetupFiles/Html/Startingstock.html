Starting stock

This screen is activated from the menu with Stock>Starting stock.

Type or scan a new product code (twice to ensure no miss-scans) and the New product screen will appear.

Pressing the 10,000 Code button will create an unused 10,000 code, including the initial 10,000.

Entering a single letter will create a new product for a Hot key. Pressing the Quit button with the mouse or pressing the Escape key returns to the sales screen.

The Starting stock screen can be used for the initial entry of stock when you are starting out, but it should not be used generally; instead use the Delivery entry screen.

When a new product is created, it will be assigned default Pack and Pack Supplier records based on the defaults set in the department chosen.
