Default stock

All products should be assigned to a department other than the default department. If any have been left assigned to the default department, or unlinked in the Product screen, they can be correctly assigned using default stock.

Select the menu item Stock>Default stock.

If there are any stock items assigned to the default department, the New product screen will appear. The details for each product can be entered or modified followed by clicking the 'OK' button. Default products are examined in reverse order of creation, and each product should be defined before moving on to the next. Items on the a,b and c keys are excluded.
