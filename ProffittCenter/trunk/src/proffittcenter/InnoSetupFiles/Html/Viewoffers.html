View offers

This table shows the current offers table entries, and is shown when you click on the menu selection Offers|Show. Useful for checking.

X is the threshold for applying an offer, and Y the discounted price. The start and end dates are just that.

If it is necessary to delete an entry, due to an error in entry, or no longer being current, select the row with a mouse, and right-click and select delete offer.

The 'Add product' button will allow further products to be added to an existing offer.

There are buttons for deleting an offer or deleting a product from an offer involving multiple products. Additional buttons allow the end date to be changed.
