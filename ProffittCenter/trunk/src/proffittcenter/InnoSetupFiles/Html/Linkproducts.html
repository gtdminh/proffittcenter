Link products

This screen allows you to link products simply. First select a department, then from the list of products shown, select the first and then click the Select button. Then click the second product and then click the second Select  button. Finally click the Link button.

Then the Link SKUs screen appears. When you link two products, there is one quantity for both; the default is to sum the two separate quantities. The figure may be changed if required. No loss will be recorded in this case.

Finally click the Link to first (or second) button. Both products will be linked to the SKU of the first (or second) SKU. Normally, choose the lower SKU number.

A single department may show too many different products to clearly see product that should be linked: the Alpha box allows for a smart search, similar to the alpha lookup function, to reduce the number of products shown. For instance, if we start with the tobacco department, typing the first two or three letters of Lambert & Butler will reduce the selection to a handful of products.
