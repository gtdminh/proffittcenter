  • Hotkeys

    A table of hotkeys in use is displayed by this menu selection. There is no new button but new hotkeys can be created by entering a letter in the Stock>Starting stock screen.

    If you need to re-assign a hotkey to a different product, look at the details, and then unlink the product. The hotkey will then appear in Stock>Default stock ready to be re-named or linked to another product.
