Packs

The Packs screen shows the basic details for pack codes. These records are created by the Delivery screen.

The box labelled Product allows the operator to just see Packs that contain a desired product by entering or scanning a product code and pressing the tab button. The same can be done for an SKU. Remember that there may be more than one pack for each product. There are buttons to delete selected Packs. As a Pack cannot be deleted when there are linked Pack supplier records, there is a separate button to delete these records first.
