Making a product permanent

When we entered new products in the sales screen, we created a temporary product, saving time when a customer is waiting. After your first rush of sales, it is time to make this temporary product into a more permanent one. Temporary products are just products that are assigned to the temporary, or default department.

To convert this temporary product to a permanent one, click on the menu selection, Stock>Default stock.

The first temporary product is shown.

Using the mouse, drop down the department combo box and select a department. There are no departments at present other than the Default one for temporary products. This time we first need to create a department for our new product. Press the Escape key to clear the screen and the select Tables>Departments for a list of available departments. Click the New button and fill in the details in the Department screen, tabbing between fields. Leave the default minimum at 0. Create an entry for a department to hold Ice Cream products, with Standard Rate Tax. When you are done, click the OK button. Press the Escape key to clear the Departments screen, as before.

Now go back to Default stock., and this time, select this new department for Ice Poles, leaving the SKU (Stock Keeping Unit) entry blank. The new SKU screen is displayed. There can be more than one product linked to the same SKU.

Entries for the new stock dialog have been filled in from the defaults for the department chosen. The stock quantity is zero, but this can be changed if you wish to reflect the true stock level.

The default settings for the ice cream department are for standard rate items (UK). If we need to change the VAT, use the drop down box. When this has been done, click the OK button with the mouse, or simply use the Tab key to step to the OK button and then press the Enter key. Then press the Enter key again to accept the remaining Product screen. If there are any more temporary products, these will appear, one by one.
