Wastes

Wastes are products that are not fit for sale. When a product is wasted, the stock levels are adjusted.

To Waste stock, when products are not fit for sale, first sell the items – multiple products or different products can be wasted at one time – and then scan the waste bar code or type 1000031 and the Enter key to complete the sale. A keyboard shortcut of ? enters the number 100003; add a 1, followed by Tab to create the code for waste.

See also returns.

The bar code for wastes can be printed out using the  option Print barcodes on the File menu.
