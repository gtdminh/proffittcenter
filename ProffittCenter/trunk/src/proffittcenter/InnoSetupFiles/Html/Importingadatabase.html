Importing a database

Accessed from the File menu, this allows data to be imported from ProffittCenter Solo. The data will be as coma separated values - a CSV file.

The Solo version has an option to export the database as a csv file using File|Database|Export to Pro. To import a file in the Pro version, select File|Import and then locate the file, normally in the My Documents folder.

After selecting the file to be imported using the conventional File Open Dialog, the file is automatically read in. The process can take some time to complete. Sales data is not imported.
