New pack

Fill in the details of the pack. A Product ID, such as a bar code, can be scanned or typed in or a search of the Products database by typing in the first few letters selecting a product with the mouse. If the product is not recognised then the New product screen will appear.

The Pack code on the first screen is the code entered in the Delivery entry screen except in the case of short codes where there is a strong chance that two manufacturers will use the same code. In this case, the code will be preceded by the Supplier ID followed by a + symbol. This creates a unique extended code for the pack; however, there is never any need to enter this extended code â€“ just stick to the short code on entry, and the extra characters will be added by the system. Pack codes are not necessarily restricted to numbers, to allow supplier stock codes to be used. These can normally be read from the invoice.

When the OK button is clicked, or the Enter key pressed, a fresh form is shown.
