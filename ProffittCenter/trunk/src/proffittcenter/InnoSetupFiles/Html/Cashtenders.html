Cash tenders

Making a cash tender

To enter cash tendered, type in the amount in pence followed by the Enter key. All values up to 1000000 (Â£10,000) are treated as cash.

There are hot keys for entering fixed values with the following mapping:

;  -  100
'  -  1000
#  - 10000
[  -  500
]  -  5000

In addition, the first three keys can be used to add zeroes to the end of a number: 12; becomes 1200 etc.

If it is found necessary to exceed this limit, press the shift key at the same time as pressing the Enter key.

If cash is to be returned to a customer after receiving on account, type 0 (zero) followed by the Enter key.
