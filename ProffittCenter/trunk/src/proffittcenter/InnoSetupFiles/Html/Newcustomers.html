New customers

Customer records are stored for one of two purposes. The first is to allow for credit and the second to keep track of customers when service or returns are a required part of the business.

New customers can be enrolled with the Customers menu selection and then clicking New. Customer numbers by default start at 1000380001, upwards. The customer number is filled in automatically.

The details of existing customers can be viewed by selecting and then clicking the Details button.

This will be used later to allow account cards to be scanned into the sales screen. The fields can be stepped through using the Tab key. Names, street names etc. are capitalised on exit. There is provision for two telephone numbers, and provision for a second card holder with a different name.

The print bar code button allows a membership card to be printed. The print is onto the default printer, using a small note size piece of paper. After being signed, this can be folded in half, trimmed with scissors, and then sealed with a credit card size laminating machine. The use of heat sealing to laminate the card stops a thermal receipt printer being used, so the cards are printed on the default printer.

The details to be filled in are self explanatory. All entries will be capitalised, so stick to lower case for maximum convenience.
