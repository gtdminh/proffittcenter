Using iReport
With the default settings for ProffittCenter, click on the Report Datasources button and create a new Database JDBC connection. I named it Till.
Select as JDBC driver, MySQL (com.mysql.jdbc.Driver)
Server Address  localhost
Database           till
Username          root
Pasword           
Tick save password, Test and if ok, Save.
Reports are saved in  'My Documents\ProffittCenterReports'
but should be saved in 'My Documents\OwnProffittCenterReports.
If not, and they have the same name, they will be overwritten by the next program install.
