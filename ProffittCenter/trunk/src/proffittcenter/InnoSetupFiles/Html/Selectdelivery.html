Select delivery

When the menu function Tables>Deliveries is activated, the first screen shown is Deliveries. This screen shows all of the Deliveries outstanding, that have not been completed. The operator can select one of these and then press the Enter key.

When a delivery is complete, tick the completed check box next to the entry. To save this, click the OK button to close the screen. Checked entries are no longer listed.

To see all deliveries, including completed ones, tick the 'Include completed' check box.
