Operator

This screen is activated from the Tables>Operators menu item and then clicking New or Details. The authority ratings run from 0 to 5.

0. The operator has full privileges â€“ reserved for owners.

1. The operator has almost the same privileges; for use by managers.

2. Senior staff member with addition rights to input deliveries.

3. Staff member with right to input deliveries.

4. Trainee staff member. Not actively allowed  to complete sales, but the sales screen is functional for training.

5. The till is locked to prevent unauthorised use.

The Print button is used to print out details, including a bar code using the default printer, suitable for laminating to make a staff pass.

Printed passes will have a preceding 1000037 so that the pass will be recognised by the .

Once you have created an additional operator, and at least one of these has owner or manager status, you can change the authority of the Default operator to 6.

In addition, you can allow any operator additional permissions by ticking the appropriate check box.

In addition, each operator can have privileges ascribed. This allows a shopkeeper to decide what activities each member of staff can perform on a case by case basis.

Managers can do all activities except modify Operators, without additional permission.
